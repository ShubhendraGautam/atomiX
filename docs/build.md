# Build and verification guide

The single, maintained command reference for building, testing, and deploying
atomiX now lives in one place:

**→ [workflow.md](workflow.md) — build, test & deploy operational reference.**

It covers profile selection, every build step, the full test surface (ISS,
cosimulation, bare-metal, roles, kernel, host-link, formal), and the ECP5
synthesis and ULX3S deployment flow.  Host setup and tool quirks are in
[dependencies.md](dependencies.md) and [toolchain.md](toolchain.md); the
physical bring-up procedure is in [ulx3s-bringup.md](ulx3s-bringup.md).
